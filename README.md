# Digster extensions

[Digster](https://github.com/frozzare/digster) extensions for Papi.

# Example

### Get field

```twig
<p>Hello {{ get_field(event, 'name') }}!</p>
```

### Get option

```twig
<p>{{ get_option('site') }}</p>
```
